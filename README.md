### About the program

#### В этом проекте реализован метод StringReverse, ринимает на вход строку и меняет порядок букв в каждом слове на обратный.

### Requirements:
#### PHP >= 8.0
#### Composer >= 2.5

### Install:
#### 1. `git clone git@github.com:Bosun18/string-reverse.git`
#### 2. `cd string-reverse`
